Your AI agent, on your machine, on the open research network.

Zoo Bot is an open agent runtime built on the Zoo research network — DeAI for decentralised intelligence, DeSci for decentralised research. Runs locally, talks on any channel, and learns from shared experience instead of weights.

Get the source → Zoo Cloud Zoo Industries 
# install the cli
$ npm i -g @zooai/bot
# run the wizard
$ zoo-bot onboard

What ships in the box

Runs locally

macOS, Linux, Windows (WSL2). Bring your own keys. Anthropic, OpenAI, Google, or local models via Ollama.

Any channel

WhatsApp, Telegram, Slack, Discord, Signal, iMessage, IRC, Matrix, Teams. DM and group chats. One agent, every surface.

Persistent memory

Becomes uniquely yours over time. Vector + relational memory, on-device by default, never sent off-machine without consent.

Tools that act

Files, shell, browser, Git, code-exec, MCP. Sandboxed by default; full-system mode when you ask for it.

Decentralised intelligence

zLLM experience-based learning over a frozen base. Training-free adaptation, governed by the Zoo DAO via ZIPs.

Open source

MIT-licensed. Issue tracker, changelog, and audits public. Foundation-stewarded — Zoo Labs Foundation 501(c)(3).
